# teamRainDrops
Objective: 
To create a painting utility using HTML, CSS and Javascript. 

Functions & Features: 

->Drawing & Styling

    1. Free Draw : using three pen styles: pen, spray pen, marker.
    2. Straight line
    3. Curve line
    4. Triangle
    5. Rectangle
    6. Circle
    7. Polygon
    8. Erase
    9. Adding images
    10. Colors : Fill and stroke
    11.Pen width.
    12.Select and cut
    13.Download and save as a png file.
